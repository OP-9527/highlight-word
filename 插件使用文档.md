# English Word Highlighter 插件使用文档

## 插件简介

English Word Highlighter 是一个功能强大的 Chrome 浏览器扩展，专门为英语学习者设计。它能够自动识别网页上的英文单词，提供翻译、发音和词汇管理功能，帮助用户高效地学习英语词汇。

## 主要功能

### 1. 单词高亮功能
- **自动模式**：自动高亮网页上所有不认识的英文单词
- **列表模式**：只高亮用户上传的自定义单词列表中的单词
- **智能识别**：使用正则表达式识别英文单词，避免误标

### 2. 翻译功能
插件集成了多个翻译源，提供全面的单词信息：

- #### 翻译源
    - **Google 翻译**：提供快速的中文翻译
    - **剑桥词典**：提供详细的英文释义和例句
    - **必应词典**：提供中文释义和发音

- #### 翻译内容
    - 单词的中文翻译
    - 英文释义和词性标注
    - 音标和发音
    - 例句和用法说明

- #### 触发方式
    - **鼠标悬停**：将鼠标悬停在高亮的单词上查看翻译
    - **文本选择**：选择任意英文文本，点击出现的搜索图标查看翻译
    - **双击选择**：双击任意英文单词快速选择并查看翻译

### 3. 发音功能
- 支持美式和英式发音
- 一键播放单词发音
- 使用 Web Speech API 提供流畅的发音体验

### 4. 词汇管理
- **已知单词标记**：点击"✔"按钮将单词标记为已认识（仅限高亮单词）
- **词汇列表管理**：在侧边栏中查看和管理已知单词
- **导入导出功能**：支持词汇列表的导入和导出
- **搜索功能**：快速查找已知单词
- **跨设备同步**：已知单词自动在所有登录同一 Chrome 账号的设备间同步

### 5. 自定义单词列表
- **文件上传**：支持上传 .txt 格式的单词列表文件
- **多文件管理**：可以上传和管理多个单词列表
- **选择性高亮**：可以选择特定文件中的单词进行高亮

### 6. 文本选择翻译功能
- **自由选择**：支持选择任意长度的英文文本（单词、短语、句子等）
- **智能识别**：自动识别包含英文字母的文本，过滤纯数字和符号
- **快速访问**：选择文本后立即显示翻译图标，点击即可查看翻译
- **便捷操作**：支持双击单词快速选择，提高使用效率

## 安装方法

### 方法一：开发者模式安装
1. 下载插件文件到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含插件文件的文件夹

### 方法二：打包安装
1. 在开发者模式下，点击"打包扩展程序"
2. 选择插件文件夹，生成 .crx 文件
3. 将 .crx 文件拖拽到扩展程序页面进行安装

## 使用方法

### 基本使用

#### 1. 启用插件
- 安装完成后，插件会自动在所有网页上运行
- 点击浏览器工具栏中的插件图标可以打开侧边栏

#### 2. 查看单词翻译
- **方式一：鼠标悬停**：将鼠标悬停在高亮的单词上
- **方式二：文本选择**：选择任意英文文本，点击出现的绿色翻译图标
- **方式三：双击选择**：双击任意英文单词快速选择并查看翻译
- 会弹出包含翻译、释义和发音的窗口
- 点击"✔"按钮将单词标记为已认识（仅限高亮单词的翻译弹窗）

#### 3. 管理已知单词
- 点击插件图标打开侧边栏
- 在"已知单词"标签页中查看所有已认识的单词
- 可以搜索、删除或导出单词列表

### 高级功能

#### 1. 自定义单词列表
1. 准备一个 .txt 文件，每行一个单词
2. 在侧边栏中点击"上传文件"
3. 选择文件并上传
4. 勾选要使用的文件，插件会只高亮这些文件中的单词

#### 2. 切换高亮模式
- **自动模式**：高亮所有不认识的英文单词
- **列表模式**：只高亮自定义列表中的单词
- 在侧边栏中可以随时切换这两种模式

#### 3. 导入导出词汇
- **导出**：将已知单词列表导出为 .txt 文件
- **导入**：从 .txt 文件导入单词列表
- 支持手动导入导出进行备份

#### 3.5. 跨设备同步（新功能）
- **自动同步**：已知单词会自动在不同设备间同步
- **使用方法**：
  1. 在所有设备上登录同一个 Chrome 账号
  2. 确保 Chrome 设置中开启了扩展程序同步
  3. 在任意设备上添加的已知单词会在 1-2 分钟内同步到其他设备
- **迁移说明**：更新到新版本后，已有的已知单词会自动迁移到同步存储
- **存储限制**：Chrome 同步存储约 100KB，大约可存储 5,000-8,000 个单词
- **配额警告**：如果单词列表过大，可能会遇到存储配额错误，建议定期导出备份

#### 4. 网站权限管理
- 可以为特定网站启用或禁用插件功能
- 在侧边栏中管理网站权限设置

#### 5. 文本选择翻译使用技巧
- **选择范围**：可以选择单个单词、短语、完整句子或段落
- **图标位置**：翻译图标会出现在选中文本的右侧
- **快速操作**：点击其他区域会自动隐藏翻译图标
- **兼容性**：与现有的高亮翻译功能完全兼容，可同时使用

## 技术特性

### 性能优化
- **CSS Highlight API**：使用现代浏览器的高亮 API，性能更优
- **事件优化**：避免重复添加事件监听器，减少内存占用
- **缓存机制**：翻译结果缓存 24 小时，减少重复请求
- **批量处理**：使用队列机制批量处理大量文本

### 兼容性
- 支持 Chrome 88+ 版本
- 使用 Manifest V3 规范
- 兼容大多数网站，包括动态内容网站

### 数据存储
- 使用 Chrome Storage API 存储用户数据
- 支持跨标签页和浏览器会话同步
- **已知单词**：使用 chrome.storage.sync 存储，支持跨设备同步（约 100KB，~5,000-8,000 词，每块 200 词）
- **自定义词汇表**：使用 chrome.storage.local 存储，设备特定（不同步）
- **插件设置**：使用 chrome.storage.local 存储，设备特定（不同步）
- 数据通过 Chrome 的安全同步机制传输，保护用户隐私

## 使用技巧

### 1. 提高学习效率
- 利用跨设备同步功能，在任何设备上无缝继续学习
- 定期导出已知单词列表作为备份
- 使用自定义单词列表针对性地学习特定领域的词汇
- 结合发音功能，提高单词记忆效果

### 2. 性能优化
- 在文本较多的页面上，建议使用列表模式而非自动模式
- 定期清理不需要的单词列表文件
- 如果页面卡顿，可以暂时禁用插件

### 3. 个性化设置
- 根据学习进度调整已知单词列表
- 为不同网站设置不同的高亮策略
- 利用搜索功能快速查找特定单词

## 常见问题

### Q: 插件在某些网站上不工作？
A: 某些网站可能因为安全策略或动态内容加载方式导致插件无法正常工作。可以尝试刷新页面或检查网站权限设置。

### Q: 翻译结果不准确？
A: 插件使用多个翻译源，如果某个源的结果不准确，可以查看其他翻译源的结果。翻译质量取决于外部 API 的服务质量。

### Q: 页面加载变慢？
A: 插件会在页面加载时处理文本内容，可能略微影响加载速度。如果影响较大，可以切换到列表模式或暂时禁用插件。

### Q: 如何备份我的词汇数据？
A: 已知单词会自动通过 Chrome 同步到云端。另外，您也可以使用侧边栏中的"导出"功能，将已知单词列表导出为文件保存作为额外备份。

### Q: 跨设备同步功能如何工作？
A: 已知单词存储在 Chrome 的同步存储中，通过您的 Google 账号自动在所有登录的设备间同步。确保在所有设备上登录同一个 Chrome 账号，并在设置中开启扩展程序同步即可。

### Q: 更新到新版本后，原有的已知单词会丢失吗？
A: 不会。插件会在首次运行时自动将已有的已知单词从本地存储迁移到同步存储，整个过程自动完成，无需任何操作。

### Q: 遇到 "quota exceeded" 错误怎么办？
A: 这表示已知单词列表超过了 Chrome 同步存储的限制（约 100KB）。解决方法：
1. 使用"导出"功能备份当前的单词列表
2. 删除一些不常用或已经掌握的单词
3. 或者继续使用导入/导出功能在设备间手动同步，不依赖自动同步
4. 建议将已知单词数量控制在 5,000-8,000 个以内

### Q: 文本选择翻译功能支持哪些类型的文本？
A: 支持选择任意包含英文字母的文本，包括单词、短语、句子、段落等。系统会自动过滤纯数字、符号或非英文内容。

### Q: 为什么有些翻译弹窗没有"✔"按钮？
A: 通过文本选择触发的翻译弹窗不会显示"✔"按钮，只有通过鼠标悬停高亮单词触发的弹窗才会显示该按钮，用于将单词添加到已知单词列表。

## 隐私说明

- 插件不会收集用户的个人信息
- 翻译请求会发送到外部 API，但不包含用户身份信息
- 已知单词通过 Chrome 的安全同步机制传输，仅在您登录的设备间同步
- 自定义词汇表和设置存储在本地浏览器中，不会同步到其他设备
- 插件不会向第三方服务器发送用户浏览数据
- 所有数据传输都通过 Chrome 的加密通道进行
